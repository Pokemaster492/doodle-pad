# Doodle Pad — Generative Kaleidoscope

A single-file, no-build browser toy for drawing symmetric, glowing particle art. Move your pointer (mouse, touch, or pen) across the canvas and it mirrors your strokes into a kaleidoscope pattern with a fading rainbow trail.

## Usage

Open [`index.html`](index.html) directly in a browser — no server or build step required.

## Controls

| Control | Effect |
| --- | --- |
| Symmetry | Number of mirrored segments (none, 2/4/6/8/12-fold) |
| Brush | Particle size |
| Trail | How quickly strokes fade |
| Rainbow | Cycle hue automatically instead of using a fixed color |
| Mirror cursor | Also mirror the reflected half of the symmetry |
| Clear | Erase the canvas |
| Save PNG | Download the current canvas as a PNG |

### Keyboard shortcuts

- `H` — show/hide the control panel
- `C` — clear the canvas
